### ⚡ EVision — EV Market Intelligence
> **Automotive · 280K records · Full ETL pipeline**  
> `Python` `PostgreSQL` `Power BI`

**The insight that mattered:**  
Reframed the EV adoption conversation — the real bottleneck isn't demand (3–4× surge post-2020, 80% BEV dominance) but **charging infrastructure**.

- Built end-to-end pipeline: raw CSV → Python cleaning & feature engineering → PostgreSQL ETL → Power BI dashboard
- Identified Tesla's **41% market share** vs second-place Chevrolet at 7% using SQL window functions (`LAG`, `ROW_NUMBER PARTITION BY`, `SUM OVER`)
- Quantified **3–4× EV adoption surge** post-2020 and **80% BEV market dominance**

### 📊 PhonePe Payment Analytics Dashboard
> **FinTech · 300K transactions · ₹3,333M volume**  
> `Power BI` `DAX`

**The insight that mattered:**  
Diagnosed **₹102M in annual failed loan value** — gave stakeholders 5 prioritised fixes they could act on immediately, no further analysis needed.

- Analysed 300K transactions across 5 service verticals using DAX measures and cross-table relationships
- Diagnosed server errors as the **#1 failure cause at ~34%** across all verticals
- Quantified **₹102M in failed Loan value** (4.05% failure rate on ₹2,532M total)
- Delivered 5 prioritised business recommendations mapped directly to failure data
